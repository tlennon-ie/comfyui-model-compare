## 🎯 The 4 Nodes

### 1. Model Compare Loaders
Define how many models you want to compare:
- Checkpoints (0-10)
- VAEs (0-5)
- Text encoders (0-5)
- LoRAs (0-10)

### 2. Model Compare Loaders Advanced
Select specific models and LoRA strengths:
- Choose which checkpoint, VAE, encoder, and LoRA for each slot
- Set LoRA strength values (e.g., "0.5, 1.0, 1.5")
- Automatically generates all combinations

### 3. Sampler Compare
Samples across all combinations:
- Loads each model configuration
- Applies LoRAs with specified strengths
- Generates output images

### 4. Grid Compare
Creates customizable comparison grid:
- Arranges images in visual grid
- Customizable borders, colors, fonts
- Saves grid + optionally individual images
- Timestamped output directories

## 🌟 Special Features

### Smart Label Generation
Automatically creates readable labels like:
```
"ckpt:model_v1 | vae:vae_a | lora_detail:0.50 | lora_boost:1.00"
```

### Flexible Grid Layout
Automatically arranges images:
- Rows = total images ÷ checkpoint count
- Columns = checkpoint count
- Customizable spacing and borders

### Combination Computing
Uses Cartesian product for all combinations:
```python
# Example:
# 2 checkpoints × 1 vae × (2 loras × 3 strengths each)
# = 2 × 1 × (2 × 3) = 12 combinations
```

### Timestamped Output
Every run creates unique directory:
```
output/model-compare/ComfyUI/Model_Comparison_20240115_143022/
```

## 🚨 Important Notes

⚠️ **After any changes:**
- Restart ComfyUI server
- Hard refresh browser (Ctrl+F5)
- Clear browser cache if needed

⚠️ **Model paths:**
- Checkpoints: `models/checkpoints/`
- VAEs: `models/vae/`
- LoRAs: `models/loras/`
- Must match your ComfyUI setup

⚠️ **LoRA strength format:**
- Comma-separated: `0.5, 1.0, 1.5`
- No spaces: ✗ `0.5 , 1.0`
- Correct: ✓ `0.5, 1.0`

⚠️ **Memory usage:**
- Each sample uses ~2-3 minutes GPU time (at 25 steps)
- 12+ GB VRAM recommended
- Monitor with `nvidia-smi`
