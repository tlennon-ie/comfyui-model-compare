## Implementation Guidelines

### Code Quality
- Type hints required (Python 3.8+)
- Error handling at system boundaries
- Defensive checks for config types (ensure dict before .get())
- Avoid mutation of inputs (clone models for patching)

### Configuration Handling
```python
# Always defensive
if isinstance(config, tuple):
    config = config[0]
if not config or not isinstance(config, dict):
    config = {}
combinations = config.get("combinations", [])
```

### Model Cloning for Sampling
```python
# CRITICAL: Always clone to avoid LoRA accumulation on piFlow
working_model = current_model.clone() if hasattr(current_model, 'clone') else current_model
```

### Seed Control Logic
```python
# All combinations in a run use SAME seed
# Seed control mode applies AFTER the run (for next submission)
use_seed = sampling_cfg.get("seed", 0)
if idx == 0:
    print(f"Using seed={use_seed} for all combinations (mode={seed_control_mode} applies after run)")

# Apply control per-combination for consistent behavior
if seed_control_mode == 'increment':
    current_seed = use_seed + idx
elif seed_control_mode == 'randomize':
    current_seed = random.randint(0, 2**32 - 1)
else:  # fixed
    current_seed = use_seed
```

## Common Bugs & Fixes

### UnboundLocalError: use_seed
**Problem**: Variable used before assignment in loop  
**Fix**: Initialize outside loop
```python
use_seed = 0
last_current_seed = 0
for idx, combo in enumerate(combinations):
    ...
```

### TypeError: 'tuple' object has no attribute 'get'
**Problem**: Config is tuple from chain instead of dict  
**Fix**: Defensive type check
```python
if isinstance(config, tuple):
    config = config[0]
```

### piFlow patch accumulation causing slowness
**Problem**: Multiple LoRA applications stacking patches  
**Fix**: Always clone model before patching
```python
working_model = current_model.clone()
```

### Seed widget not updating dynamically
**Problem**: STRING widgets don't update with `control_after_generate`  
**Fix**: Use INT widgets with `control_after_generate: True`
```python
"seed": ("INT", {
    "default": 0,
    "min": 0,
    "max": 2**32 - 1,
    "control_after_generate": True
})
```
