# Grid Builder Export Features

## Overview

The Grid Builder now supports exporting your edited grids to multiple formats, including high-resolution "megagrid" images perfect for presentations and documentation.

## Supported Export Formats

### 1. **HTML Grid** (✓ Always Available)
- Interactive web page with all features
- Filterable, sortable, lightbox viewing
- Self-contained with embedded images
- Shareable via URL

### 2. **PNG Image (Megagrid)** (Requires Playwright)
- High-resolution full-grid screenshot
- Perfect for presentations, documentation
- Captures the entire grid layout as a single image
- Lossless quality, larger file size

### 3. **JPEG Image** (Requires Playwright)
- Compressed image format
- Smaller file size than PNG
- Good for web sharing, smaller storage
- Configurable quality (default: 95%)

### 4. **PDF Document** (Requires Playwright + ReportLab)
- Professional document format
- Includes grid image + metadata table
- Shows statistics (image count, dimensions, parameters)
- Perfect for archiving and sharing

### 5. **CSV Data** (✓ Always Available)
- All image parameters in spreadsheet format
- Import into Excel, Google Sheets, etc.
- Useful for analysis and data processing

### 6. **JSON Config** (✓ Always Available)
- Complete grid configuration
- Row/column hierarchy definitions
- Styling and metadata
- Machine-readable format

## Installation

### Base Installation
The Grid Builder works out of the box with HTML, CSV, and JSON exports.

### Enable Image/PDF Export
To enable PNG, JPEG, and PDF exports, install these optional dependencies:

```bash
# Navigate to the plugin directory
cd custom_nodes/comfyui-model-compare

# Install Python packages
pip install playwright reportlab

# Install Chromium browser for rendering
playwright install chromium
```

**That's it!** Once installed, the PNG, JPEG, and PDF export options will automatically become available.

## Usage

### From Grid Editor

1. Open a grid in the Grid Builder (click "Edit Grid" from gallery)
2. Make your changes (hierarchy, labels, styles, etc.)
3. Click the **Export** button (top-right)
4. Choose your desired format:
   - **HTML Grid** - Interactive web page
   - **PNG Image (Megagrid)** - High-res full grid screenshot
   - **JPEG Image** - Compressed image
   - **PDF Document** - Professional document with metadata
   - **CSV Data** - Spreadsheet data
   - **JSON Config** - Configuration file
5. Enter a name for your export
6. Wait for the export to complete

### From Generated Grids

All HTML grids generated by the Model Compare node include an **Export** button:

1. Open any generated grid HTML file
2. Click the **Export** button in the top-right corner
3. Select your format
4. The file will be saved to the same directory

## Export Options Explained

### Image Exports (PNG/JPEG)

**How It Works:**
- Uses Playwright to render the HTML in a headless browser
- Captures a full-page screenshot at 1920px width
- Hides UI elements (buttons, checkboxes, menus)
- Produces a clean, presentation-ready image

**Image Dimensions:**
- Width: 1920 pixels (configurable in code)
- Height: Automatic based on grid content
- Typical file sizes:
  - PNG: 5-20 MB (lossless)
  - JPEG: 1-5 MB (compressed)

**Best For:**
- **PNG**: Presentations, documentation, when quality matters
- **JPEG**: Web sharing, email attachments, storage-conscious uses

### PDF Export

**How It Works:**
- Renders the grid to a PNG image using Playwright
- Creates a PDF with ReportLab
- Embeds the grid image
- Adds a metadata table with grid information

**PDF Contents:**
1. **Grid Image** - Full rendered grid visualization
2. **Metadata Table** - Title, creation date, image count, format
3. **Statistics Table** - Cell counts, dimensions, parameters

**Best For:**
- Archival and documentation
- Sharing via email (single file)
- Printing high-quality comparisons
- Professional reports

## Troubleshooting

### "Image rendering not available" Error

**Cause:** Playwright not installed

**Solution:**
```bash
pip install playwright
playwright install chromium
```

### "PDF export requires Playwright" Error

**Cause:** ReportLab or Playwright not installed

**Solution:**
```bash
pip install playwright reportlab
playwright install chromium
```

### Export Takes a Long Time

**Normal Behavior:**
- HTML rendering: Instant
- PNG/JPEG: 5-15 seconds (browser rendering)
- PDF: 10-20 seconds (includes PNG rendering + PDF creation)

**Large Grids:**
- Grids with 50+ images may take 20-30 seconds
- This is normal - the browser needs to load all images
- You'll see progress messages in the console

### Export Button Doesn't Respond

**Check:**
1. Make sure the grid has fully loaded
2. Check browser console for errors (F12)
3. Try refreshing the page
4. Verify ComfyUI server is running

## Advanced Configuration

### Change Image Resolution

Edit `grid_renderer.py` to change the viewport width:

```python
# Default: 1920px wide
viewport_width: int = 1920

# High-res: 3840px (4K)
viewport_width: int = 3840

# Medium: 1280px
viewport_width: int = 1280
```

### Change JPEG Quality

Edit `grid_builder_routes.py`:

```python
# Default: 95% quality
quality=95

# Maximum quality: 100%
quality=100

# Smaller files: 85%
quality=85
```

### Customize PDF Layout

Edit `export_service.py` to modify PDF styling, add sections, or change page size.

## Tips & Best Practices

### For Best Image Quality
- Use **PNG** for lossless quality
- Export at 1920px or higher resolution
- Ensure all images are loaded before export

### For Smallest File Size
- Use **JPEG** at 85% quality
- Or use **HTML** + browser's built-in screenshot feature

### For Professional Documents
- Use **PDF** export
- Include descriptive titles in your grids
- Configure labels and subtitles before exporting

### For Data Analysis
- Export to **CSV** for spreadsheet analysis
- Use **JSON** for programmatic processing

## Performance Notes

**Export Speed:**
- HTML/CSV/JSON: Instant (<1 second)
- PNG/JPEG: 5-15 seconds
- PDF: 10-20 seconds

**Memory Usage:**
- Large grids (100+ images) require more RAM
- Playwright uses ~200-500 MB during rendering
- Temporary files are cleaned up automatically

**Disk Space:**
- PNG: Largest (5-20 MB per grid)
- JPEG: Medium (1-5 MB per grid)
- PDF: Medium (2-10 MB per grid)
- HTML: Small (500 KB - 2 MB per grid)
- CSV/JSON: Tiny (<100 KB per grid)

## FAQ

**Q: Do I need to install Playwright for HTML export?**
A: No, HTML export works out of the box.

**Q: Can I export multiple grids at once?**
A: Not yet, but you can export each grid individually from the gallery.

**Q: Why is the PNG export so large?**
A: PNG is lossless compression. Use JPEG for smaller files or export HTML and screenshot.

**Q: Can I export just a portion of the grid?**
A: Not currently. You can crop the exported image in an image editor.

**Q: Does export work offline?**
A: Yes, once Playwright is installed, no internet connection is needed.

**Q: Can I customize the export button?**
A: Yes, edit the HTML templates in `html_grid_generator.py` and `grid-editor.html`.
